How to protect files in HDFS

To protect files in HDFS, you can use two methods:
  • Access control list (ACL)
  • File permissions
Check the info below where we try to explain how these methods are working.

File permissions

In HDFS, you can restrict access to files or directories via a standard model - POSIX.

Access control list

Access control list - is a list with access permission for files or/and directories. ACL grants us a more flexible and visual view of how to assign permission for HDFS objects.
You can use ACL if you need set permission for the different users (group) on a same file or directory, as an example.
  1. Add the below property in the hdfs-site.xml file: 
    
                  <property>
                    <name>dfs.namenode.acls.enabled</name>
                    <value>true</value>
                    </property>
