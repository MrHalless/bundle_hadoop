Data storage software
Arenadata Hadoop
Arenadata Hadoop (ADH) is a full-fledged enterprise distribution package based on Apache Hadoop and designed for storing and processing semi-structured and unstructured data.
Arenadata Database
Reliable and scalable enterprise data storage using massively parallel processing and built on Open Source Greenplum DBMS, keeping pace with your business growth. ADB is designed to store, process, and analyze large amounts of structured and semi-structured data.
Arenadata Grid
Arenadata Grid (ADG), developed jointly with Mail.Ru and based on Tarantool technology, is a distributed in-memory computing system enabling real-time data processing as part of the unified enterprise data platform.
Data streaming and processing software
Arenadata Streaming
Arenadata Streaming (ADS) is a scalable, fault-tolerant solution for real-time streaming data processing, adapted for enterprise and built on Apache Kafka and Apache Nifi.
Arenadata QuickMarts
If you need to promptly analyze large data volumes while also reducing the load of your data storage system, Arenadata Quickmarts (ADQM) is a perfect solution for you.
